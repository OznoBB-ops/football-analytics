# Football Analytics

Аналитическая система для поиска валуйных ставок в футболе.

## 🌐 Веб-интерфейс
**https://ozno-football.streamlit.app** (после настройки кастомной ссылки)

## 📊 Возможности
- Поиск личных встреч (H2H)
- Валуйные ставки (edge >= 10%)
- Статистика команд
- Графики трендов
- Анализ тоталов по лигам

## 🤖 Telegram-бот
- `/analyze` — анализ линии БК
- `/values` — валуйные паттерны
- Поиск H2H по командам
- Подписки на команды

## 📈 База данных
11000+ матчей из 7 лиг:
- FIN (Финляндия) — ROI +21%
- POL (Польша) — ROI +5%
- TU1 (Турция), SC1 (Шотландия), B1 (Бельгия), G1 (Греция), P1 (Португалия)

## 🔄 Автообновление
